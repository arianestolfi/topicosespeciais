p5.sound extende p5.js com funcionalidades de Web Audio functionality incluindo audio input, playback, analyse, and sinthese.

## pasta topicos especiais

É o exemplo vazio de um website. Inclui o arquivo do site, index.html, e um arquivo base para começar os sketches.

### index.html

index.html é o arquivo base para o site, ele já importa as bibliotecas necessárias para trabalhar. 

### sketch.js

The sketch.js é o modelo para o sketch em p5.js sketch, com as funções setup() e draw() para serem completadas.
